Goker: Go + Poker
05:40:55 PM 9/12/2020
Goker is a project for my Technology class (3ème, France) (Equivalent of 9th grade in the USA), written in Go, it incorporates the chehsunliu/poker library for solving combinations fast. Goker is also a challenge for me, I love to learn new things and this project will consolidate my Go knowledge. Go is one of my favorite languages because it is clean, fast, and relatively mature. Goker's interface is very simple and clean, it is modular and compartmentalized. There are some useful features, like auto-reconnect, and all clients are identified by IP addresses to prevent multiple browsers being used to have multiple accounts. As a whole, I am very proud of Goker and I hope you will enjoy it.
